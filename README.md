# C Shell

Very simple C Shell project to exercise the following C Topics:

- pointers
- fork, exec
- functions, scopes
- heap, malloc
- arrays, strings
- string manipulation
- passing by value, passing by pointer
- Makefiles, targets, dependencies
- C Compile Process

## Requirements

- Basic Linux knowledge
- GCC

## Usage

~~~~
make
./bin/code
# Exit it with Ctrl + C 
make clean
~~~~
