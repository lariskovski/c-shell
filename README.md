# C Shell

Very simple C Shell project to exercise the following C Topics:

- pointers
- fork, exec
- functions, scopes
- heap, malloc
- arrays, strings
- string manipulation
- passing by value, passing by pointer
- Makefiles

## Requirements

- Basic Linux knowledge
- GCC

## Usage

~~~~
make build
make run
# Exit it with Ctrl + C 
make clean
~~~~
