# C Shell

Very simple C Shell project to exercise the following C Topics:

- pointers
- fork, exec
- functions, scopes
- heap, malloc
- arrays, strings
- string manipulation
- passing by value, passing by pointer

## Requirements

- Basic Linux knowledge
- GCC

## Usage

~~~~
gcc code.c -o code -lreadline
./code
~~~~
